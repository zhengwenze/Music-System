# SingerController 接口返回数据格式

## 重要说明
- 歌手模块(Mod_singer)仅使用自己模块的实体类，不使用其他模块的实体类
- 具体来说，音乐相关功能使用的是`org.zwz.mod_singer.entity.Music`实体类

## 接口基本信息
- 模块名称：Mod_singer
- 端口号：8087
- 基础路径：`http://localhost:8087/singer`

## 接口列表

### 1. GET /singer/all - 获取所有歌手信息
**示例请求URL：** `http://localhost:8087/singer/all`

## 响应数据结构说明

### Singer 对象字段说明
| 字段名 | 类型 | 说明 |
|-------|-----|-----|
| id | Integer | 歌手ID |
| username | String | 用户名 |
| password | String | 密码（加密存储） |
| email | String | 邮箱 |
| phone | String | 手机号 |
| role | Integer | 角色（1表示歌手） |
| activation | Integer | 激活状态（0激活，1禁用） |
| createTime | String | 创建时间（ISO格式） |
| imageUrl | String | 头像URL |
| about | String | 歌手简介 |

### Mess 对象字段说明
| 字段名 | 类型 | 说明 |
|-------|-----|-----|
| code | Integer | 状态码（200表示成功） |
| message | String | 响应消息 |
| data | Object | 响应数据（可选） |
